# JT7 — Master Index

## Purpose
JT7 is an operator system for job-search execution.

It is not a dashboard.

It helps the user:
- detect signals
- prioritize action
- execute work
- track pipeline
- learn from outcomes

## Core Model
Signals → Today’s Plan → Execution → Manage → Intelligence

## Modes
EXECUTE
- Today’s Plan

MANAGE
- Jobs
- Recruiters
- Outreach
- Messages

INTELLIGENCE
- Competition
- Wiki
- Reports

SYSTEM
- Design System
- Settings
