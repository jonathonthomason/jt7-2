# JT7 — Master Index

## Purpose
JT7 is an operator system for job-search execution.

It is not a dashboard.

It helps the user:
- detect signals
- prioritize action
- execute work
- track pipeline
- learn from outcomes

## Core Model
Signals → Today’s Plan → Execution → Manage → Intelligence

## Modes
EXECUTE
- Today’s Plan

MANAGE
- Jobs
- Recruiters
- Outreach
- Messages

INTELLIGENCE
- Competition
- Wiki
- Reports

SYSTEM
- Design System
- Settings

## Build Order
1. Shell + routing
2. Review Queue
3. trusted-state model wiring
4. Pipeline Overview
5. Opportunity Detail
6. Design system
7. Remaining manage surfaces
8. Intelligence layer
