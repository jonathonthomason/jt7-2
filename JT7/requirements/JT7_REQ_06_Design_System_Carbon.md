# JT7 — Design System (Carbon)

## Base
IBM Carbon Design System

## Tone
- dark
- low-glare
- dense
- functional

## Color
- Blue → action
- Green → success
- Yellow → attention
- Red → blocked
- Purple/Cyan → system
- Gray → neutral

- Section 24/30
- Card 18/24
- Body 16/24
- Label 12/16
